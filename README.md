# Warodai Parser

Неофициальный парсер словаря [Warodai](https://warodai.ru).
Создан для удобного извлечения и струтурирования информации из словарных статей,
для последующего конвертирования в словари других форматов, автоматического создания
карточек для Anki и прочего.

## Использование

- Установите python версии 3.11+.
- Склонируйте репозиторий  или скачайте и разархивируйте его.
- Скачайте словарь с сайта [Warodai](https://warodai.ru) ([прямая ссылка](https://warodai.ru/download/warodai_txt.zip)).
- Распакуйте словарь (txt файл) в директорию с кодом проекта.
- Запустите парсер: `python warodai_parser.py`
- В директории должен появиться файл `warodai_out.json`.

*Планируется добавление других форматов, помимо json.*
